# Property Maintenance DAO

A decentralized autonomous organization for coordinating property maintenance in apartment complexes and HOAs, built on the Stacks blockchain.

## Overview

This DAO enables transparent, efficient, and democratic property maintenance management through smart contracts. Residents can submit maintenance requests, contractors can bid on work, and expenses are tracked and distributed fairly among property owners.

## Core Features

### 🏠 Maintenance Requests
- Submit maintenance requests with priority levels
- Photo/document attachments for issues
- Resident voting on request priorities
- Automated approval workflows

### 💼 Contractor Bidding
- Transparent bidding process for maintenance work
- Contractor verification and rating system
- Automated bid evaluation and selection
- Performance tracking and reviews

### 💰 Expense Tracking
- Real-time expense monitoring and reporting
- Fair cost distribution among property owners
- Budget allocation and spending controls
- Financial transparency and auditability

## Key Benefits

- **Democratic Decision Making**: Residents vote on priorities and major decisions
- **Cost Transparency**: All expenses tracked and auditable on-chain
- **Fair Distribution**: Costs distributed based on property ownership/square footage
- **Quality Assurance**: Contractor ratings and performance tracking
- **Efficient Process**: Automated workflows reduce administrative overhead

## Getting Started

### Prerequisites
- Clarinet CLI installed
- Node.js and npm
- Property ownership verification
- Basic understanding of Clarity smart contracts

### Installation

1. Clone the repository
2. Install dependencies: `npm install`
3. Check contracts: `clarinet check`
4. Run tests: `npm test`
